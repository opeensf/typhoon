# 杰瑞的台风工作台

**Jerry's Typhoon Workbench** 是一个面向气象爱好者和台风追踪者的个人桌面工作台。它把追台时常用的论坛、卫星云图、雷达图、官方预报页面、天气图和参考资料集中到一个界面里，减少在多个浏览器标签页之间反复切换的麻烦。

这个应用不是普通天气 App，而是一个偏专业、偏个人化的追风信息工作台。它适合在台风活跃期集中查看多个网页、保存常用研判场景，并快速切换不同资料组合。

## 功能特点

- 支持 1 / 2 / 3 / 4 屏浏览模式
- 三屏模式采用左侧大窗格、右侧两个小窗格的布局
- 左侧气象标签栏支持分类、收藏、置顶、改名和拖拽
- 支持导入自定义网页
- 支持顶部快捷模式，可新增、编辑、重命名和删除
- 支持把当前布局保存为快捷模式
- 每个标签可设置独立自动刷新间隔
- 窗格支持加载状态和失败提示
- 支持外部浏览器打开、复制链接、刷新、最大化和关闭窗格
- 内置 NMC 各高度层天气图入口
- 内置速度转换和云图参考悬浮图

## 内置资源

应用内置整理了常用追台资源，包括：

- 台风论坛、台风吧、TY_Board、STORM2K
- JTWC、NMC、HKO、海南气象等官方机构页面
- RAMMB、NICT 向日葵、风云四号、Digital Typhoon 等卫星云图
- Zoom Earth、Earth Nullschool、Ventusky、Meteologix 等环境场工具
- NMC 地面、925hPa、850hPa、700hPa、500hPa、200hPa、100hPa 天气图

## 运行方式

安装依赖：

```powershell
npm.cmd install
```

构建：

```powershell
npm.cmd run build
```

运行：

```powershell
npm.cmd start
```

Windows 下也可以直接双击：

```text
run-workbench.cmd
```

## 开发模式

```powershell
npm.cmd run dev
```

## 说明

部分网站可能会限制嵌入式浏览器显示。遇到这类页面时，可以使用窗格右上角的外部打开按钮。

本项目默认不爬取、不下载中央气象台天气图，仅整理官方页面入口，方便在工作台中集中访问。
